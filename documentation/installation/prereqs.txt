System Requirements
===================


Basic Requirements

Basic requirements to run Pjango are:

  -Web server
  -Server read/write access for the working web directory
  -PHP engine
  -.htaccess engine


Recommended Settings

It is recommended that you have the following:

  -PHP >=5.3.0
  -SQL server access
  -UNIX/LINUX environment


Developer Tools

If you wish to develop new modules, tags, filters, etc. for
Pjango, you should have the following:

  -Makefile engine
