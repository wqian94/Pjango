class ValComponent

extends AbstractComponent

The ValComponent class represents a variable and the filters applied to it.

Additionally, the comment tag block {# (comment here) #} is also represented as a special case of the ValComponent.

Static Variables

const TYPE -- Has value of ValComponent::VAL.

Instance Variables

public components -- Array of the children of this component.

protected filters -- A queue of filters to be applied to this variable.

protected name -- Name/identifier of the tag.

Functions

__construct($tok) -- The Token to represent. Automatically parses for name and filters. Initializes the uninitialized class variables.

getFilters() -- Returns the array of filters passed into this tag.

getName() -- Returns the name of this tag.

render($args) -- Renders this tag to generate some string output, based on the variables defined in $args. Each key in $args represents the identifier of the variable to be used.
