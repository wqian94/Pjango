class RootComponent
===================


extends AbstractComponent

 
The RootComponent class is the container component class, which
represents the root of the entire component tree.

If $root is the name of the root component, then calling

"$root->render($args)"

will render the entire template, and

"$root->getRendered()"

returns the rendered output string.
 
Instance Variables
 
public components -- Array of the children of this component.

private rendered -- Initialized null. The rendered output
  string generated by the call to render($args).
 
Functions
 
__construct()
  -- Initializes instance variables.

getRendered()
  -- Returns the rendered output string generated by
     render($args).

getName()
  -- Returns the name of this tag.

render($args)
  -- Renders this tag to generate some string output, based on
     the variables defined in $args. Each key in
     $args represents the identifier of the variable to be
     used. The final string is stored in the rendered class
     variable.
