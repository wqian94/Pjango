SQL

The SQL library is provided to make SQL querying more seamless and convenient. Most notable is the SQL class, used to help organize SQL handles, as well as to provide an easy transition from PHP's mysql_* to the newer mysqli_* API.

However, any settings, such as hostnames, usernames, passwords, and database selections should be done in settings.php, using the $SQL_DATABASES variable.
