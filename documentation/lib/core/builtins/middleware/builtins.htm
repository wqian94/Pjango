Built-in Middleware Reference

References for the built-in middleware are found below.

AutoTrimMiddleware

Preprocessing: None.

Rendering: Removes all carriage returns (\r), line feeds, (\n), and tabs (\t) from the input.

CsrfMiddleware

Preprocessing: If POSTDATA exists, checks CSRF token for security. Otherwise, no action.

Rendering: None. Returns inputted text.
