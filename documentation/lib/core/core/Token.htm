class Token

The Token class is the most primitive breakdown of the template file. The Token itself is a container of the raw text it represents (content) and the identified type of that raw content (type).

Static Variables

const TAG -- Has value "tag".

const TXT -- Has valule "txt".

const VAR -- Has value "var".

Instance Variables

private content -- The raw string that this Token represents.

private type -- The type of the content represented by this Token. Takes on values of Token::TAG, Token::VAL, and Token::TXT.

Functions

__construct($str) -- Constructor function, takes in content string and assigns the the appropriate type to type.

getContent() -- Returns the raw content string for this Token.

getType() -- Returns the type of this Token -- the value of type.
